# 용어집 (출입/방문자 도메인)

> 코드·문서·커밋에서 같은 용어를 쓰기 위한 사전. 새 용어가 생기면 여기 추가.

| 용어 | 의미 |
|------|------|
| 출입(Access) | 게이트/도어를 통과할 권한 및 행위 |
| 도어/디바이스(Door/Device) | BiostarX 가 제어하는 물리 출입 장치 |
| 권한 부여/회수 | 특정 대상에게 출입 권한을 주거나 거두는 행위 (감사 대상) |
| 방문자(Visitor) | 상시 출입자가 아닌 임시 출입 대상 |
| 사전등록 | 방문 전 방문자 정보를 미리 등록 |
| 발급/회수(Issue/Revoke) | 방문자 출입수단(카드/권한) 발급·회수 (감사 대상) |
| 이력/감사(Audit) | 민감 행위의 기록 (누가/무엇을/언제/대상/결과) |
| 어댑터(Adapter) | BiostarX 등 외부 연동을 격리하는 계층 |
| BiostarX | Suprema 사의 출입통제 플랫폼 (외부 연동 대상) |
